# 刷记 Release Notes

## v1.5.6 (2026-06-19)

### 修复

- **修复文件夹外键的数据库 schema 不一致**：此前 `cards` 表的外键声明在实体与迁移脚本之间
  不一致，可能导致 (1) 从旧版本升级的用户启动时崩溃；(2) 删除文件夹时该文件夹下的卡片未能
  正确归入「未分类」。现已统一（数据库升级至 v7），并新增迁移回归测试保障。

### 优化

- 新增 CI（GitHub Actions）：每次推送自动跑 ktlint 与单元测试；补充刷卡计数 / 年费自动续期
  等核心流程的单元测试。
- 修正「选择语言」对话框样式：选项不再因 `ListItem` 的实心表面色在深色主题下显示为
  对话框内一块更黑的卡片；改用透明、带单选无障碍语义的行，与对话框表面协调统一。
- 移除一处历史误提交的空 `CreditRecord` 子模块占位与少量死代码；内部一致性改进。
- 新增 `docs/Design.md` 设计文档与根目录 `CLAUDE.md` 开发说明。

## v1.5.5 (2026-06-18)

### 修复

- 修复切换应用内语言时界面出现黑屏 / 白屏闪烁的问题：语言切换触发的 `recreate` 间隙现以淡入淡出过渡掩盖，切换更平滑。

## v1.5.4 (2026-06-18)

### 修复

- 修复自定义主题色取色器：再次打开时停留在当前已选颜色，而不是每次回到默认色。
- 修复主页「未分类」分组标题在英文界面下仍显示中文的问题。
- 修复卡片详情页「刷卡记录」的笔数被显示成「N / N」的文案错误，现正确显示为刷卡笔数。

### 优化

- 主页整体进度统计改用缓存计算，卡片较多时滚动更流畅。
- 启动续期逻辑收口到应用容器内部，移除一处脆弱的向下类型转换；清理冗余 DAO 命名、未使用的字符串资源与无效的文本溢出配置。
- 内部代码质量与一致性改进。

## v1.5.3 (2026-06-18)

### 新增

- **应用内语言切换**：在「设置 → 语言」可在「跟随系统 / 简体中文 / English」之间切换，切换后即时生效，并由系统记忆所选语言。
- 新增完整英文界面（English），全部界面文案均已本地化。

### 优化

- 外观设置各项补全图标，视觉更统一。
- 全量界面文案（含导入 / 导出的错误提示）抽取为字符串资源，为后续多语言扩展打好基础。
- 内部代码质量改进。

## v1.5.2 (2026-06-17)

### 修复

- **自定义主题色现在会作用于整个界面**。此前选择自定义颜色时只有按钮和强调文字会变色，背景与卡面仍停留在默认的蓝绿色调；现已修复，背景、各级表面与容器颜色都会随所选颜色一同协调变化。

### 优化

- 自定义配色改用 Material 3 官方动态取色方案生成整套颜色，观感与系统动态色保持一致、过渡更自然。
- 内部稳定性与代码质量改进。

## v1.5.1 (2026-06-17)
